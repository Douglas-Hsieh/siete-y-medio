This program is supposed to emulate a real life game called “siete y medio.”
The game plays a game of “siete y medio” with you, and keeps a log of the rounds that you played with the dealer. When you run the program, important information should be kept in the gamelog.txt.

Contents:

siete_y_medio.cpp: contains the main routine and code to execute the game

cards.cpp: contains implementation code for classes in cards.h

cards.h: contains the Player, Hand, and Card class and respective member function and field declarations
